# 流行语数据录入工具

一个简易的 Flask 前端，用于方便地录入流行语数据。

## 界面预览

```
┌──────────────────────────────────────────────────────────────────┐
│  流行语数据录入工具                           已有词条：1 个    │
├────────────────────────┬─────────────────────────────────────────┤
│                        │                                         │
│   📄 示例参考           │   📝 录入表单                           │
│                        │                                         │
│   ┌──────────────┐     │   ┌─────────────────────────────────┐   │
│   │ 选择词条 ▼   │     │   │ 基本信息                        │   │
│   └──────────────┘     │   │  词条ID / 词目 / 语言 / 词性    │   │
│                        │   └─────────────────────────────────┘   │
│   {                    │                                         │

│       "language":      │   └─────────────────────────────────┘   │
```

## 快速开始

### 1. 安装依赖

```bash
cd data_entry
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

### 3. 访问页面

打开浏览器访问 http://localhost:5000

## 功能特性

- **左侧示例区**：显示已有词条的 JSON 结构，可切换查看不同词条
- **右侧录入区**：表单化录入，自动根据语言显示对应发音字段
- **预览功能**：提交前预览生成的 JSON
- **自动更新**：提交后自动更新 `buzzwords.json` 和 `js/main.js`

## 支持的语言

| 语言 | 发音字段 |
|------|---------|
| English | UK / US (IPA) |
| Chinese | 拼音 |
| Japanese | 罗马字 / 平假名 |
| Korean | 罗马化 / 韩文 |
| 其他 | IPA / 罗马化 |

## 注意事项

1. 确保从 `data_entry` 目录运行 `python app.py`
2. 提交前请使用「预览 JSON」功能检查数据格式
3. 词条ID 建议使用词条本身（小写），中文可直接使用中文
4. 词频数据至少添加 1 年的数据
